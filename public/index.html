GitHub to Discord Notification Service

About

This service listens for GitHub webhook events and forwards them to a Discord channel using webhooks.

Notifications are sent for various GitHub events including:

  • Issues: opened, closed, reopened, assigned, labeled, etc.
  • Pull requests: opened, closed, merged, reopened, ready for review

Each notification includes:

  • Clickable links to issues and pull requests
  • Information about who created, closed, or modified the item
  • Assignee and reviewer details
  • Formatted messages with appropriate emojis

Endpoints

/github - GitHub webhook endpoint

Setup Instructions

To use this service:

  1. Create a webhook in your Discord channel settings
  2. Add the webhook URL to the Vercel environment variables
  3. Configure your GitHub repository to send webhook events to https://your-deployment-url.vercel.app/github
